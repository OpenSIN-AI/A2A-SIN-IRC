# A2A-SIN-IRC

> A2A agent for IRC integration within the OpenSIN ecosystem.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Stars](https://img.shields.io/github/stars/OpenSIN-AI/A2A-SIN-IRC)](https://github.com/OpenSIN-AI/A2A-SIN-IRC)

## Overview

This repository provides an Agent-to-Agent (A2A) integration for IRC, enabling seamless communication and automation within the OpenSIN-AI ecosystem.

## Quick Start

```bash
git clone https://github.com/OpenSIN-AI/A2A-SIN-IRC.git
cd A2A-SIN-IRC
npm install
npm start
```

## Features

- A2A protocol support for IRC
- Automated message routing
- Integration with OpenSIN MCP servers
- Full observability via OpenSIN-Ledger

## Architecture

This agent connects to the OpenSIN orchestrator and handles IRC-specific messaging through the A2A protocol.
